上述为Win11（伪）云电脑的体验

可以听歌，浏览（与正常浏览器无异，Google不能访问懂的都懂，Bing和Edge会自动同步本地账号，ps：我可没偷窥你密码和ip），看空文件夹（毕竟是假的，没有存储空间）。

至于语言？

Sorry，I don‘t know.

反正不是Markdown这种小儿科的东西。

cin>>"我是智障";

cout<<"QS";（？？？笔者早忘了）

这种C++主导的时代正在进行时，也正在过期中。

老牌的HTML，JAVA，JavaScript,Python,C#仍老当益壮。

而C++的大展拳脚其实在游戏领域（确信）。

新进崛起的易语言，甚至Scratch等图形化编程工具以及某些国内高校天天嚷嚷着搞事情的，最适合我这种文科生的，遥遥无期 而又牵人心弦 的中文语言编程，当然不可避免的用到了英文缩写(可谓掩耳盗铃)。

总之，在你的Blog中实现这样一个funny的小PLUGIN并非Obstacles（讽刺某些语言和人）。

Key就在于如何deploy好。

详情可以自行百度iframe是什么；iframe怎么用；Node.js框架支持iframe吗 ……

最重要的，你要develop到Github fork codes 的 ability。

git clone https://github.com/Skykine/Skykine.Github.io.git

(示例)

git pull git push 当然别忘了提前配置SSHkeys

自己搭建一个优秀项目的镜像会更方便。

二级域名一抓一大把。

300冤永久的服务器一抓一大把。

Freenom甚至提供了免费一级域名。

你需要做的只有到**云和##云将你的GitHub Forked先部署到服务器上，再绑定域名（实名≥18），最后备案（劝退向）。别忘了购买8000远永久的ssl证书，并注册百度站长，一套流程fc方便。

最后，在编辑你的post（*/**/index.html)时，记得加上iframe。

All in all，我不会告诉你Serverless会发挥多大的作用的。
